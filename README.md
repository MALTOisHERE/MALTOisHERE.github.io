# Personal Portfolio Website

This is a personal portfolio website for Mohamed AIT OUAARAB, a cybersecurity engineer. The website showcases his skills, experience, and projects. It is a single-page application built with HTML, CSS, and vanilla JavaScript.

## Features

- **About Me**: A brief introduction to Mohamed AIT OUAARAB.
- **Resume**: A summary of his education and work experience.
- **Skills**: A list of his technical skills.
- **Portfolio**: A gallery of his projects (currently disabled).
- **Contact**: A contact form with a Google reCAPTCHA.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/MALTOisHERE/maltoishere.github.io.git
   ```
2. Open `index.html` in your browser.

## Usage

The website is a static website. All the content is in `index.html`. The website is navigated by clicking on the links in the navigation bar. The contact form uses Formspree to send emails. You will need to replace the form action URL with your own Formspree URL. You will also need to replace the Google reCAPTCHA site key with your own site key.
